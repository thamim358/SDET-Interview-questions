## 📖 Overview

The **SDET Master Study Kit** is a comprehensive, free, open-source interview preparation resource for Software Development Engineers in Test (SDET), QA Automation Engineers, and Test Architects. It contains **380+ high-value interview questions** across **11 topic areas**, curated from real interview patterns at top tech companies.

Every question includes detailed answers, code examples in JavaScript/TypeScript, interview strategies, and priority ratings to help you focus your study time effectively.

> 💡 **For the best experience**, open the HTML files in Chrome and press `Ctrl+P` → "Save as PDF" to create premium-looking PDF study guides with the built-in dark theme.

## 🚀 How to Use

### Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/sdet-master-study-kit.git
   cd sdet-master-study-kit
   ```

2. **Open the dashboard**
   - Open `SDET-MASTER-STUDY-KIT/index.html` in **Google Chrome** for the best experience
   - Browse topics, search questions, and navigate the roadmap

3. **Start studying**
   - **P1 questions first** — these appear in nearly every interview
   - **P2 next** — for breadth and confidence
   - **P3 for senior roles** — deep-dive topics for Staff/Lead positions

### Creating PDF Study Guides

This kit is designed to create beautiful, print-optimized PDFs:

1. Open any HTML file in **Google Chrome**
2. Press **`Ctrl+P`** (Windows) or **`Cmd+P`** (Mac)
3. Set destination to **"Save as PDF"**
4. Under **More settings**:
   - Paper size: **A4**
   - Margins: **None**
   - Check **"Background graphics"** (preserves the dark theme)
5. Click **Save** — your premium PDF is ready!

## 📅 Study Plan

### Suggested Learning Path

| Phase | Topic | Duration | Weight |
|-------|-------|----------|--------|
| 1 | JavaScript | 2-3 weeks | 🟢 Foundation |
| 2 | TypeScript | 1-2 weeks | 🟢 Foundation |
| 3 | Playwright | 4-6 weeks | 🔴 Core |
| 4 | API Testing | 2 weeks | 🔴 Core |
| 5 | SQL + Database | 2 weeks | 🔴 Core |
| 6 | System Design | 2-3 weeks | 🟠 Senior |
| 7 | DevOps | 2-3 weeks | 🟠 Senior |
| 8 | Framework Architecture | 2 weeks | 🟠 Senior |
| 9 | Mobile Automation | 2-3 weeks | 🟡 Niche |
| 10 | AI Testing | 2 weeks | 🟡 Niche |
| 11 | Interview Prep | 3-4 weeks | 🏁 Final |

### Study Tips

- **🎯 Start with P1**: Master all P1 (red) questions first — they cover 80% of interview scenarios
- **🔄 Progressive Learning**: JavaScript → TypeScript → Playwright → Framework Design, then branch out
- **📝 Practice Aloud**: Read the "Real Interview Answer" sections and practice saying them aloud
- **💬 Prepare STAR Stories**: Have 7-8 behavioral stories ready with quantified impact
- **🗺️ Follow the Roadmap**: Open `SDET-Master-Roadmap.html` for detailed phase-by-phase guidance
