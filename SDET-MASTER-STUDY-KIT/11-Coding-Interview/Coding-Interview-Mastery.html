🟢 Easy — Frequently Asked

1
Two Sum — Given an array and target, return indices of two numbers that sum to target.Easy
Solution

Use a hash map to store complements while iterating. For each number, check if target - num exists in map.

function twoSum(nums, target) { const map = new Map(); for (let i = 0; i < nums.length; i++) { const complement = target - nums[i]; if (map.has(complement)) return [map.get(complement), i]; map.set(nums[i], i); } return []; }
Complexity

Time: O(n)Space: O(n)Pattern: Hash Map, Complement Lookup

🎯 SDET Relevance Most commonly asked problem. The complement pattern is useful for comparing API response data against expected values.

2
Valid Parentheses — Determine if a string of brackets (){}[] is valid.Easy
Solution

Use a stack. Push opening brackets, pop and match for closing brackets. Stack must be empty at end.

function isValid(s) { const stack = []; const pairs = { ')': '(', ']': '[', '}': '{' }; for (const c of s) { if ('([{'.includes(c)) stack.push(c); else if (stack.pop() !== pairs[c]) return false; } return stack.length === 0; }
Complexity

Time: O(n)Space: O(n)

🎯 SDET Relevance Stack pattern directly applies to HTML/XML validation in web testing and JSON structure verification.
3
Valid Palindrome — Check if a string is a palindrome considering only alphanumeric chars.Easy
Solution

Two-pointer approach. Skip non-alphanumeric characters, compare case-insensitively.

function isPalindrome(s) { let l = 0, r = s.length - 1; while (l < r) { while (l < r && !/[a-zA-Z0-9]/.test(s[l])) l++; while (l < r && !/[a-zA-Z0-9]/.test(s[r])) r--; if (s[l].toLowerCase() !== s[r].toLowerCase()) return false; l++; r--; } return true; }
Complexity

Time: O(n)Space: O(1)

4
Valid Anagram — Determine if two strings have the same characters with same frequency.Easy
Solution

Use frequency counter. Count chars in first string, decrement for second. All counts should be zero.

function isAnagram(s, t) { if (s.length !== t.length) return false; const count = {}; for (const c of s) count[c] = (count[c] || 0) + 1; for (const c of t) { if (!count[c]) return false; count[c]--; } return true; }
Complexity

Time: O(n)Space: O(1)

🎯 SDET Relevance Frequency counter pattern is used to compare list contents regardless of order — common in test data validation.
5
Binary Search — Implement binary search on a sorted array (iterative and recursive).Easy
Solution

Divide and conquer. Compare middle element with target, eliminate half the search space each iteration.

function binarySearch(nums, target) { let l = 0, r = nums.length - 1; while (l <= r) { const m = Math.floor((l + r) / 2); if (nums[m] === target) return m; if (nums[m] < target) l = m + 1; else r = m - 1; } return -1; }
Complexity

Time: O(log n)Space: O(1)

🎯 SDET Relevance Binary search is fundamental for efficient test data lookup and divide-and-conquer debugging strategies.
6
Contains Duplicate — Check if any value appears at least twice in an array.Easy
Solution

Use a Set. If adding an element finds it already exists, return true.

function containsDuplicate(nums) { const seen = new Set(); for (const n of nums) { if (seen.has(n)) return true; seen.add(n); } return false; }
Complexity

Time: O(n)Space: O(n)

🎯 SDET Relevance Directly applicable to checking for duplicate test data, verifying unique constraints in UI validation.
7
Reverse String — Reverse a string in-place (array of characters).Easy
Solution

Two-pointer approach. Swap characters at left and right pointers, move inward.

function reverseString(s) { let l = 0, r = s.length - 1; while (l < r) { [s[l], s[r]] = [s[r], s[l]]; l++; r--; } }
Complexity

Time: O(n)Space: O(1)

8
FizzBuzz — Classic screening question. Print Fizz, Buzz, or number for 1 to n.Easy
Solution

Check divisibility by 15 first (3 AND 5), then 3, then 5, else the number. Order matters!

function fizzBuzz(n) { return Array.from({length: n}, (_, i) => { i++; if (i % 15 === 0) return 'FizzBuzz'; if (i % 3 === 0) return 'Fizz'; if (i % 5 === 0) return 'Buzz'; return String(i); }); }
⚠️ Trap Check FizzBuzz FIRST (divisible by both). Many candidates check Fizz first, which catches numbers divisible by 15 too.
9
Climbing Stairs — Count distinct ways to climb n steps (1 or 2 steps at a time).Easy
Solution

Dynamic programming. ways[n] = ways[n-1] + ways[n-2] (Fibonacci). Optimize to O(1) space with two variables.

function climbStairs(n) { if (n <= 2) return n; let a = 1, b = 2; for (let i = 3; i <= n; i++) { const c = a + b; a = b; b = c; } return b; }
Complexity

Time: O(n)Space: O(1)

10
Merge Sorted Array — Merge two sorted arrays into one sorted array in-place.Easy
Solution

Two-pointer approach from the end. Compare elements from the last positions and place in correct position.

function merge(nums1, m, nums2, n) { let i = m - 1, j = n - 1, k = m + n - 1; while (j >= 0) { if (i >= 0 && nums1[i] > nums2[j]) nums1[k--] = nums1[i--]; else nums1[k--] = nums2[j--]; } }
Complexity

Time: O(m+n)Space: O(1)

11
Remove Duplicates from Sorted Array — Remove duplicates in-place, return new length.Easy
Solution

Two-pointer technique. One pointer for unique elements, one for iteration. When values differ, move unique pointer forward and copy.

function removeDuplicates(nums) { if (nums.length === 0) return 0; let k = 0; for (let i = 1; i < nums.length; i++) { if (nums[i] !== nums[k]) { k++; nums[k] = nums[i]; } } return k + 1; }
Complexity

Time: O(n)Space: O(1)

12
First Unique Character — Find first non-repeating character in a string.Easy
Solution

Two-pass approach. First pass: count frequencies. Second pass: find first character with count === 1.

function firstUniqChar(s) { const count = {}; for (const c of s) count[c] = (count[c] || 0) + 1; for (let i = 0; i < s.length; i++) { if (count[s[i]] === 1) return i; } return -1; }
Complexity

Time: O(n)Space: O(1) (26 letters, or Unicode if extended)

🟡 Medium — Commonly Asked

13
Longest Substring Without Repeating Characters — Sliding window.Medium
Solution

Sliding window with two pointers and hash map tracking last seen index. When duplicate found, move left past previous occurrence.

function lengthOfLongestSubstring(s) { let max = 0, left = 0; const map = new Map(); for (let right = 0; right < s.length; right++) { const char = s[right]; if (map.has(char)) left = Math.max(left, map.get(char) + 1); map.set(char, right); max = Math.max(max, right - left + 1); } return max; }
Complexity

Time: O(n)Space: O(min(m,n))

🎯 SDET Relevance Sliding window pattern is useful for analyzing dynamic content extraction and handling paginated data streams.
14
Maximum Subarray (Kadane's Algorithm) — Find contiguous subarray with largest sum.Medium
Solution

Track current sum and max sum. Reset current sum if it becomes negative. Kadane's algorithm.

function maxSubArray(nums) { let curr = nums[0], max = nums[0]; for (let i = 1; i < nums.length; i++) { curr = Math.max(nums[i], curr + nums[i]); max = Math.max(max, curr); } return max; }
Complexity

Time: O(n)Space: O(1)

15
Reverse a Linked List — Reverse a singly linked list (iterative and recursive).Medium
Solution

Iterative: track prev, current, next pointers. Recursive: reverse rest, then point next.next to current.

function reverseList(head) { let prev = null, curr = head; while (curr) { const next = curr.next; curr.next = prev; prev = curr; curr = next; } return prev; }
Complexity

Time: O(n)Space: O(1) iterative

16
Product of Array Except Self — Return products of all elements except self, without division.Medium
Solution

Prefix and suffix products in two passes. First pass: left products. Second pass: multiply by right products.

function productExceptSelf(nums) { const result = new Array(nums.length).fill(1); let left = 1; for (let i = 0; i < nums.length; i++) { result[i] = left; left *= nums[i]; } let right = 1; for (let i = nums.length - 1; i >= 0; i--) { result[i] *= right; right *= nums[i]; } return result; }
Complexity

Time: O(n)Space: O(1)

17
Linked List Cycle — Detect cycle using Floyd's Tortoise and Hare algorithm.Medium
Solution

Two pointers: slow moves one step, fast moves two steps. If they meet, there's a cycle.

function hasCycle(head) { let slow = head, fast = head; while (fast && fast.next) { slow = slow.next; fast = fast.next.next; if (slow === fast) return true; } return false; }
Complexity

Time: O(n)Space: O(1)

🎯 SDET Relevance Floyd's algorithm tests ability to detect infinite loops — directly applicable to debugging flaky tests and detecting circular refs in data.
18
Group Anagrams — Group strings that are anagrams of each other.Medium
Solution

Use a hash map keyed by sorted string or character count. Group strings with same key together.

function groupAnagrams(strs) { const map = new Map(); for (const s of strs) { const key = s.split('').sort().join(''); if (!map.has(key)) map.set(key, []); map.get(key).push(s); } return [...map.values()]; }
Complexity

Time: O(nk log k)Space: O(nk)

🎯 SDET Relevance Grouping pattern applies to categorizing test results, grouping API responses, and clustering similar failure patterns.
19
Move Zeroes — Move all zeroes to end while maintaining relative order of non-zero elements.Medium
Solution

Two-pointer approach. Maintain position for next non-zero element, swap when found.

function moveZeroes(nums) { let pos = 0; for (let i = 0; i < nums.length; i++) { if (nums[i] !== 0) { [nums[pos], nums[i]] = [nums[i], nums[pos]]; pos++; } } }
Complexity

Time: O(n)Space: O(1)

20
Three Sum — Find all triplets that sum to zero. No duplicate triplets.Medium
Solution

Sort array, then fix one element and use two-pointer technique on the rest. Skip duplicates.

function threeSum(nums) { nums.sort((a,b) => a - b); const result = []; for (let i = 0; i < nums.length - 2; i++) { if (i > 0 && nums[i] === nums[i-1]) continue; let l = i+1, r = nums.length-1; while (l < r) { const sum = nums[i] + nums[l] + nums[r]; if (sum === 0) { result.push([nums[i], nums[l], nums[r]]); while (l < r && nums[l] === nums[l+1]) l++; while (l < r && nums[r] === nums[r-1]) r--; l++; r--; } else if (sum < 0) l++; else r--; } } return result; }
Complexity

Time: O(n²)Space: O(1)

21
Implement a JSON Validator — Check if a string is valid JSON.Medium
Solution

Use try/catch with JSON.parse. For deeper validation, use schema validation (like AJV) that checks structure matches expected schema.

function isValidJSON(str) { try { JSON.parse(str); return true; } catch { return false; } } // Validate schema compliance function validateSchema(data, schema) { for (const [key, type] of Object.entries(schema)) { if (typeof data[key] !== type) return false; } return true; }
🎯 SDET Relevance Essential for API response validation. Every test automation framework needs JSON validation utilities.

23
Implement Custom Retry Logic with Exponential Backoff (async version).Medium
Solution

Retry with exponential backoff: wait baseDelay × 2^attempt between retries. Include jitter to avoid thundering herd.

async function retryWithBackoff(fn, maxRetries = 3, baseDelay = 500) { for (let i = 0; i < maxRetries; i++) { try { return await fn(); } catch (err) { if (i === maxRetries - 1) throw err; const delay = baseDelay * Math.pow(2, i) + Math.random() * 100; // +jitter await new Promise(r => setTimeout(r, delay)); } } }
🎯 SDET Relevance Directly applicable to handling flaky elements, network timeouts, and dynamic content loading in test automation.
24
Merge Intervals — Merge overlapping intervals.Medium
Solution

Sort intervals by start time. Iterate — if current interval overlaps with previous, merge them. Otherwise add previous to result.

function merge(intervals) { intervals.sort((a,b) => a[0] - b[0]); const result = [intervals[0]]; for (let i = 1; i < intervals.length; i++) { if (intervals[i][0] <= result[result.length-1][1]) { result[result.length-1][1] = Math.max(result[result.length-1][1], intervals[i][1]); } else { result.push(intervals[i]); } } return result; }
Complexity

Time: O(n log n)Space: O(n)

🎯 SDET Relevance Merge intervals pattern is useful for scheduling test execution windows, analyzing overlapping time ranges in test data.
25
Promise.all Implementation — Implement Promise.all from scratch.Medium
Solution

Create a new Promise. Iterate inputs, resolve each. If any rejects, reject immediately. If all resolve, resolve with array of results.

function promiseAll(promises) { return new Promise((resolve, reject) => { const results = []; let completed = 0; if (promises.length === 0) resolve([]); promises.forEach((p, i) => { Promise.resolve(p).then(val => { results[i] = val; completed++; if (completed === promises.length) resolve(results); }).catch(reject); }); }); }
Complexity

Time: O(n)Space: O(n)

🎯 SDET Relevance Tests deep understanding of async patterns. Promise.all is used for parallel test fixtures and concurrent API validations.
26
Debounce vs Throttle — Implement both from scratch.Medium
Solution

Debounce: timer-based, resets on each call. Throttle: timestamp-based, executes at most once per interval.

function debounce(fn, delay) { let timer; return function(...args) { clearTimeout(timer); timer = setTimeout(() => fn.apply(this, args), delay); }; } function throttle(fn, interval) { let lastCall = 0; return function(...args) { const now = Date.now(); if (now - lastCall >= interval) { lastCall = now; fn.apply(this, args); } }; }
27
Flatten Nested Array — Flatten an array of arbitrary nesting depth (implement Array.flat).Medium
Solution

Recursive approach: iterate through array, if element is array, recurse. Or use stack-based iterative approach.

function flatten(arr, depth = Infinity) { return arr.reduce((acc, val) => acc.concat(Array.isArray(val) && depth > 0 ? flatten(val, depth - 1) : val), []); }
Complexity

Time: O(n)Space: O(n)
